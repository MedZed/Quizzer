# Quizzer
Save quiz questions with answers in Json file (node)

## Exemple : 
```json
[
    {
        "question":"What's the tallest building in the world",
        "correct":"Burj khalifa",
        "falseOne":"Burj al arab",
        "falseTwo":"Ifel toure",
        "falseThree":"The great wall of China"
    }
]
```
